Interactive Scenario Builder

Scenario Self Generation Agent

질문에 순서대로 답하면 사용자에게 더 매력적으로 다가가는 앱 시나리오를 단계적으로 완성합니다.

Flow

  • 접근 확인 후 질문이 순서대로 열립니다.
  • 각 답변 직후 방향을 넓혀주는 인사이트가 제공됩니다.
  • 완료 후 역할별 활용안과 보고서 다운로드가 가능합니다.

Access

시작하기

현재 화면은 입력 UI만 제공하며 실제 검증은 서버 연동 후 활성화됩니다.

Step Flow

Scenario Setup

지역에 따라 추천 기기와 활용 방식이 달라집니다.

연령, 생활 패턴, 계절, 집 안의 문제 상황까지 함께 적으면 더 설득력 있는 시나리오가 나옵니다.

복수 선택으로 구성하고, 핵심 기기 조합을 기준으로 시나리오를 만듭니다.

답변을 마칠 때마다 다음 단계를 더 흥미롭게 만드는 힌트가 여기에 표시됩니다.

Output

Scenario Output

접근 확인 후 네 가지 질문을 순서대로 완료하면 상세 시나리오와 역할별 활용안을 확인할 수 있습니다.
