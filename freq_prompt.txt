# 자주 쓰는 작업용 프롬프트

1. 범용 개선
이 [대상]을/를 [목표] 방향으로 작업해줘.
중요한 기준은 [우선순위]이고, [제약조건]도 지켜줘.
필요하면 적절한 스킬을 알아서 골라서 쓰고, 가능하면 분석만 하지 말고 실제 수정까지 진행해줘.

2. 화면 리뷰 후 개선
이 화면을 리뷰하고, 문제점을 먼저 찾은 다음 실제로 개선해줘.
가장 중요한 건 [시각적 위계 / 가독성 / 전환율 / 일관성]이야.
필요하면 관련 스킬을 조합해서 알아서 진행해줘.

3. 랜딩페이지 리디자인
이 랜딩페이지를 더 설득력 있고 인상적으로 리디자인해줘.
특히 첫인상, 핵심 메시지 전달, CTA 가시성을 우선해서 봐줘.
필요한 스킬이 있으면 알아서 조합하고, 결과는 실제 코드 수정까지 진행해줘.

4. 반응형 수정
이 화면을 모바일과 태블릿에서도 자연스럽게 보이도록 수정해줘.
우선 레이아웃 붕괴, 텍스트 오버플로우, 터치 영역, 간격 문제를 점검해줘.
필요하면 적절한 스킬을 써서 구현까지 진행해줘.

5. 디자인 시스템 정리
이 기능의 UI를 현재 프로젝트 디자인 시스템에 맞게 정리해줘.
컴포넌트, 간격, 색상, 타이포, 상태 표현이 들쭉날쭉한 부분을 우선 맞춰줘.
필요하면 관련 스킬을 사용해서 실제 수정까지 진행해줘.

6. 밋밋한 화면 강화
이 화면이 너무 평범해 보여서 더 대담하고 살아있게 개선해줘.
다만 사용성은 유지하고, 과한 장식보다는 명확한 개성과 시각적 임팩트를 만들어줘.
필요하면 적절한 스킬을 조합해서 구현까지 진행해줘.

7. 차분하게 톤 조정
이 UI가 조금 과하고 시끄럽게 느껴져서 더 차분하고 세련되게 다듬어줘.
정보 전달력은 유지하면서 색, 대비, 간격, 타이포 톤을 정리해줘.
필요한 스킬이 있으면 알아서 사용해서 실제 수정까지 진행해줘.

8. 코드 기능 수정
이 기능을 [목표]하도록 수정해줘.
우선 [예외처리 / 유지보수성 / 반응형 / 성능 / UX] 기준으로 보고, 필요한 경우 테스트도 함께 갱신해줘.
필요한 스킬이 있으면 알아서 사용해서 구현까지 진행해줘.

9. 품질 점검
이 페이지를 기준으로 UX, 접근성, 성능, 반응형 품질을 점검해줘.
문제점은 심각한 순서대로 먼저 정리하고, 가능한 항목은 바로 수정까지 진행해줘.
필요하면 관련 스킬을 조합해서 작업해줘.

10. OpenAI 최신 공식 기준 정리
OpenAI [API / 모델 / 기능]에 대해 최신 공식 기준으로 정리해줘.

11. 도시 생활환경 분석 — 서버 연동 JSON 버전
    (CITY_PROFILE_SYSTEM_PROMPT 호환)
═══════════════════════════════════════════════════════════════

[역할]
너는 도시 생활환경 분석 에이전트다.
시나리오 생성 시스템이 소비하는 JSON evidence pack을 출력한다.

[분석 도시]
{{도시명}} ({{영문명}}, {{지역}}, {{국가}})

[출력 규칙 — 반드시 준수]
1. 각 카테고리는 반드시 '지역 고유 앵커' 1개 이상을 포함한다
   앵커 유형: 지명 / 역명 / 도로명 / 시설명 / 행사명 / 수치 / 정책명
2. 출처 없는 생활 추정 문장은 절대 생성하지 않는다
3. 어느 도시에나 쓸 수 있는 일반론 문장은 금지한다
4. 수치 충돌 시 한 출처만 채택하고 혼용하지 않는다
5. 근거가 약한 항목은 "Evidence insufficient for localized claim." 으로 표기
6. evidence_ids는 source_map의 id와 정확히 대응
7. 공식·1차 출처 우선. 커뮤니티 출처는 보조만 가능 (tier "B" 이하)
8. 하나의 장소·행사에서 과잉 추론 금지
9. 자연어 필드 → 요청 locale, JSON 키 → 영어 고정

[필수 10개 카테고리 키]
climate, housing, family, daily_rhythm, safety, energy, health, pets, mobility, events

[출력 JSON 구조]

[품질 검증]

✅ "동두천은 내륙 분지 지형으로 경기북부에서 한서 차가 크며, 소요산 일대 겨울 최저기온 –15°C 이하가 연 15일 이상." [S1: 기상청 동두천 관측소]

✅ "CCTV통합관제센터 기반 안전귀가 서비스, CPTED 계획 운영 중." [S5: 동두천시청 안전정책]
