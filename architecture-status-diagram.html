현재 vs 계획 비교 다이어그램

요청하신 6개 팩트를 기준으로 AS-IS(현재)와 TO-BE(계획)를 나란히 정리했습니다.

AS-IS (현재)

Live

Backend

서버/API

External

Anthropic API

팩트 6: 정적 서빙에서 /.env 차단 미명시로 키 노출 위험이 있습니다.

No AI API Cost
