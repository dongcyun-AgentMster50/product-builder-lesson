# AI 플랫폼 비교 검증 프로젝트

## 과제명
**시나리오 셀프 생성 에이전트 — AI 플랫폼 비교 검증**

## 과제 개요
지역 특성 및 타겟 세그먼트, 반영 기기 정보를 기반한 최적 시나리오 발굴 및 확장/활용

## 비교 대상
| 플랫폼 | 버전 | 역할 |
|---|---|---|
| Claude (v2.0) | Sonnet 4.6 | **Baseline (기준)** |
| OpenAI Codex CLI | GPT-4o | 비교 대상 A |
| Google Gemini | AI Studio / Flash | 비교 대상 B |

## 진행 일정
| 주차 | 기간 | 내용 |
|---|---|---|
| 1주차 | Week 1 | 환경 구축 및 계획 수립 |
| 2주차 | Week 2 | 집중 테스트 |
| 3주차 | Week 3 | 성과 측정 및 비교 검증 |
| 4주차 | Week 4 | 최종 평가 및 리포팅 |

## 폴더 구조
```
ai-platform-comparison/
  week1/   → 환경 세팅, 요구사항 정의, TC 목록, Baseline 결과물
  week2/   → Codex / Gemini 테스트 결과 및 채점
  week3/   → 3-Way 비교 분석, 차트, 인사이트 초안
  week4/   → 최종 리포트, 발표 PPT, Lesson Learned
```

## 핵심 테스트 케이스 (TC)
- **TC-01** 기본 시나리오 생성 (리테일 / 한국·서울 강남 / Care)
- **TC-02** 멀티 제약 이행 (브랜드 / 미국·뉴욕 / Save + Secure)
- **TC-03** 초개인화 (TC-01 + 아파트·3인가족·에어컨 보유)
- **TC-04** 채널 확장 (TC-01 + 카피라이팅·이메일)
- **TC-05** 코드 재현 (SmartThings 시나리오 생성 웹앱 재현)
